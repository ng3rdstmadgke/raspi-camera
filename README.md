# Picamera2 Install

```bash
sudo apt-get update
sudo apt-get upgrade

# picamera2: https://pypi.org/project/picamera2/
# ※ picamera2をpipでインストールするのは非推奨らしい
sudo apt install -y python3-pyqt5 python3-opengl
sudo apt install -y python3-picamera2
```
